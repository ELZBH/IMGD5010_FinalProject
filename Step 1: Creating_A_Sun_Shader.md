# Sun Shader

Using a fragment and vertex shader allows us to give a wobbly effect to shapes. Using this, I set a yellow circle in the center of a blue screen. Though this evoked the sun, I wanted to make the project more dynamic. Thus, I added a second shader and a smaller circle atop the first. This resulted in a more layered image that better evoked the bending of light rays. Below is a sample of the halfway point.
