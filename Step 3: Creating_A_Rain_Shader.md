Above is a sample of a few rows of ellipses generated in the Rain Shader code. These shapes are individual raindrops. Each row is assigned one of two shaders (with slightly different behaviors and noticeably different hues)
and set a small distance below (increase in y value) the previous row. This generated a set of many "waving" blue ellipses that look vaguely like rain. 

# Setting the scene

Modifying the first version of the cloud shader helped further develop the scene for the rain sahder. It also obscured parts of certain rows, which made the rain feel even more dynamic. 

You can review the result at this link: https://editor.p5js.org/ELZBH/full/l3QJdvnMa
